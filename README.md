# before-after-component

PROJECT: Before and After Image Comparison Component
This custom web component is built for use on a construction website to showcase a project before and after. 
LIVE DEMO: https://lewis-0870.github.io/before-after-component/

PROJECT OVERVIEW:
Purpose - to allow users to compare before and after photos of paving/construction/masonry work.
Goal - create an easily adaptable, reusable, high-impact visual tool for a professional construction website.

FEATURES:
* Use of Shadow DOM to prevent issues with existing CSS.
* Responsive design. Adapts to different screen sizes using CSS styles
* Custom Attributes - Easily adaptable via before and after attributes
